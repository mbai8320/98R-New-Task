HTML Elements
 Semantic Elements

Elements that describes about the content are called semantic elelments.
Examples:

  • h1 to h6 all heading tags
  • strong tag
  • i tag
  • em tag
  • p tag and etc..

Non-Semantic Elements

Elements that should not describe the content are called non-semantic elements.
Examples:

  • div tag
  • span tag
  • bold tag

Block Level Elements

It occupies complete the block where it start and where it ends.
Examples:

  • p tag

Inline Elements

It will just take the content space.
Examples:

  • del tag
  • em tag and etc..
